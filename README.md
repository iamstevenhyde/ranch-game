## Play the game

### [https://iamstevenhyde.github.io/ranch-game/](https://iamstevenhyde.github.io/ranch-game/)

Open that link on any phone or laptop. No GitHub account needed. One person taps
**Host a Game** to start it and reads out the 5-letter join code; everyone else taps
**Join a Game** and enters that code.

Classroom strategy game for MBA 546 (Boise State). Instructor console: admin.html.
